# Dashboards de Vendas, Finanças, RH e Logística

## Objetivo
Criar insights em dashboards para análise dos dados de diferentes áreas.

## Insights do Dashboard de Vendas
- análise mensal e anual de faturamento e qtd. vendida;
- quais marcas venderam mais e as que venderam menos;
- faturamento total;
- produto mais vendido;
- faturamento por continente.
